# Como Redigir e Preparar um Artigo Científico

## Definições Base
- **Pesquisa Científica** (NBR 6022): "Processo formal e sistemático de desenvolvimento do método científico. O objetivo fundamental é descobrir respostas para PROBLEMAS mediante o emprego de procedimentos científicos."
- **Artigo Científico** (NBR 6022): "É parte de uma publicação com autoria declarada, que apresenta e discute ideias, métodos, técnicas, processos e resultados nas diversas áreas do conhecimento."

Tipos: revisão de literatura, estudo de caso, aplicação de modelo, estudo comparado, estudo documental.

## Estrutura Geral do Artigo

```
PARTE 1: Elementos Pré-textuais
PARTE 2: Elementos Textuais
PARTE 3: Elementos Pós-textuais
```

---

## PARTE 1 — Elementos Pré-textuais

### Ordem dos Autores
- **Primeiro autor** = principal contribuidor, quem escreveu mais
- Autores do 2º em diante: importância diminui conforme posição
- **Último autor** = geralmente o orientador (pode ter importância maior)

### Como Elaborar o Resumo
1. Apresente, mas não explique
2. Corte palavras desnecessárias
3. Verifique as informações
4. Finalize com o começo
5. **Palavras-chave**: funcionam como "tags" — ajudam a categorizar a produção acadêmica

---

## PARTE 2 — Elementos Textuais

### Estrutura IMRaD
| Seção | Conteúdo | Pergunta-chave |
|-------|----------|----------------|
| **Introdução** | Informações sobre tema, justificativa, objetivo | De que trata? Por que foi feita? O que se sabia antes? |
| **Método** | Tipo de estudo, cenário, amostra, procedimentos, aspectos éticos | Como o estudo foi realizado? |
| **Resultados** | Achados + análise estatística | O que foi encontrado? |
| **Discussão** | Interpretação, comparações, conclusão | O que significam os achados? O que este estudo acrescenta? |

---

## PARTE 3 — Elementos Pós-textuais

- **Referências bibliográficas**: formatação conforme normas da revista/periódico
  - Ferramenta recomendada: [Zotero](https://www.zotero.org/)
- **Anexo(s)**: documentos que não são do autor e servem de fundamentação (estatísticas, questionários validados, mapas, leis, figuras)
- **Apêndice(s)**: material elaborado pelo próprio autor para complementar argumentação

## Checklist para Artigo Bem Escrito
1. [ ] Decida o objetivo do artigo
2. [ ] Escolha o periódico para submissão
3. [ ] Use guias que facilitem a redação (ex: template SBC)
4. [ ] Escreva a estrutura primeiro
5. [ ] Complemente o texto com as partes faltantes
6. [ ] Revise diversas vezes
7. [ ] Certifique-se de que está metodologicamente correto
8. [ ] Assegure-se de que não há falhas de redação
9. [ ] Submeta para publicação
10. [ ] Lide adequadamente com editores e revisores

## Anti-Plágio
> "Um artigo é um documento científico. Não pode ser cópia de outro já publicado e nem plágio, sujeito às penalidades do Direito Autoral brasileiro e mundial."
